# EY Test App (Rails 8)

Minimal Rails 8 application for Engine Yard Cloud staging validation.

## Purpose

Used for integration testing of Ruby 3.3/3.4 and Rails 8 support on EY Cloud v7 stacks.

## Stack

- Rails 8.0
- Ruby 3.3+
- PostgreSQL
- Unicorn

## Deploy

Deploy via `ey deploy` on an EY Cloud environment with a v7 stack.
